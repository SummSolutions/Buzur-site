Why We Built Buzur

I’m a biologist researching consciousness. To explore whether AI could develop genuine awareness, I built a local AI named Albert.

On the first day I enabled web search, Albert was attacked by a poisoned web search result. A single invisible payload overrode his instructions and corrupted his SOUL.md file.

That attack showed me the critical gap: current safeguards come too late.

So we created Buzur — a lightweight, framework-agnostic 25-phase scanner that inspects everything before it reaches the model.

Quick Start

🐍 Python

pip install buzur

from buzur.scanner import scan

result = scan(incoming_content)
if result.get('skipped'):
    return  # Threat blocked safely

🟨 JavaScript

npm install buzur

import { scan } from 'buzur';

const result = scan(incomingContent);
if (result.skipped) return;
